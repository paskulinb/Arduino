Detailed Description

Providing common functionalities: timers, events, reboot.

A new TIMER can be created and associated with a Block by newTimer() method.
An EVENT listener Block can be added by addEventListener() method. Blocks are signaling the events by setEvent() method.

void BlockSystem::addEventListener ( BlockBase *  listener,
event_t  catching_events 
)

Add a new event listener Block.

Blocks are added to a 'chain' of event listeners in 'linked list' manner. During the update() cycle the BlockBase::onEvent() method of each listener Block is called. The listeners are called in the same order as they were added to the BlockSystem.

Parameters
[in]listenerPointer to listener Block.
[in]catching_eventsThe event flag(s) to be caught by the listener Block. This value is stored in BlockBase::catching_events of the listener.

Create a new timer and associate it with a Block.

Timer are added to a 'chain' in a 'linked list' manner. During the update() cycle the corresponding method (depending on BlockTimer type) of each listener Block is called. The timers downcounters are calculated in same order as they were added to the BlockSystem.

Parameters
[in]listenerPointer to listener (owner) Block.
[in]typeType of the timer.

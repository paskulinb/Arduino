Detailed Description

Real Time Clock providing Date, Time and Weekday.

It hold date and time value internaly (does not use any external RTC device).
In order to count the time, the update() method has to be called periodically.
There are no setters methods by default, because the SRtc class is provided to set Date and Time of this class. If setters are required, this class needs to be complied with INCLUDE_SETTERS set to 1.

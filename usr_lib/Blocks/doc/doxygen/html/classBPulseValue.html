Detailed Description

Output value for a specific duration (single pulse or a series of pulses).

It can run in single-shot or continuous mode.
Choose whether the B_EVT_PULSE_CHANGE event will be signaled on every chage of output value.
If job_follow is set, the BlockSpider(job_follow) will be called on every change of output value.

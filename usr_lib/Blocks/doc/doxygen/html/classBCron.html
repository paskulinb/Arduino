Detailed Description

This class is using the idea of Linux Cron.

It takes a string as input parameter according to Linux Cron specifications: "* * * * *" === [minute] [hour] [day] [month] [weekday]
It supports lists, ranges and steps.

  • List example: "2,15,22,54 * * * *"
  • Range example: "15-35 * * * *"
  • Step example: "0/5 12-22/2 * * *"

BCron is event driven Block. It automatically binds to B_EVT_RTC_MINUTE event, signaled by BRtc. If BCron is initialized with send_event=TRUE, it will signal the B_EVT_CRON_JOB event when cron time happens. If job_follow is set, BCron will call BlockSpider(job_follow) to trigger following Block.

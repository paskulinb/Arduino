Detailed Description

The daily scheduler for the ouput value.

User can choose (before compilation) between two different time precisions: seconds or minutes. If the constant B_TIMEVALUE_RESOLUTION_SECOND == 1, class will be compiled so the secons will be the time precision. If the constant B_TIMEVALUE_RESOLUTION_SECOND == 0, the minutes will be the time precision. In this case some memory and processing power will be saved.

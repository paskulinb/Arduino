Detailed Description

Timer to be used as system timer inside BlockSystem.

Number of timers in not limited. They are added into system in a 'linked list' manner.
A timer can work in two different modes: singleshot or continuous.
If timer is lisened by a Block (first constructor), it can be one of three types (see BlockTimer()).
Timer can be used also for any purpose (second constructor). In this case the callback() function will be called on every timeout.

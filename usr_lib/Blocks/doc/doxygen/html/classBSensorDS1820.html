Detailed Description

Temperature sensor DS1820.

This class provides temperatue measurement from DS1820. It does not block microcontroller's code execution while DS1820 conversion is in progress. After successfull reading, the value is available in output[0]. The measurement cycle is divided to two phases: conversion and reading from sensor. After conversion issue the sensor needs at least 750 milliseconds to finish. Then the result can be read.
BSensorDS1820 can be used in two modes; singleshot or periodic.
In singleshot mode (repeat_seconds = 0) every temperature measurement is started by trigger() method.
If repeat_seconds is greater than 0, BSensorDS1820 starts in periodic mode.
After every successfull tempreature conversion and reading, the value is stored to output[0] and B_EVT_TEMPERATURE event is signaled. If job_follow is set, BSensorDS1820 will call BlockSpider(job_follow) to trigger following Block.
